# aicut-skill

Codex skill collection for AI-assisted ecommerce video editing workflows.

## Skills

- `outdoor-ecommerce-video-mixer`: 户外运动装备电商短视频批量混剪、豆包 ASR 转写、VectCut/剪映草稿生成、字幕/卖点/关键词/气口/黑屏质检工作流。

## Install

Copy the skill folder into your Codex skills directory:

```powershell
Copy-Item -Recurse .\skills\outdoor-ecommerce-video-mixer $env:USERPROFILE\.codex\skills\
```
